
lightestdark's portfolio workspace is built with Next.js App Router and tailored for a technical dark, animation-first single-page experience.

## Stack

- Next.js 14 (App Router)
- Tailwind CSS 3
- GSAP + ScrollTrigger (`gsap`, `@gsap/react`)
- Lenis smooth scrolling (`lenis`)
- Framer Motion (`framer-motion`)
- Google Fonts via `next/font` (Syne + IBM Plex Mono)

## Development

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

```bash
npm run build
npm run start
```

## Notes

- If `node`, `npm`, or `npx` are not recognized in PowerShell, restart your terminal after Node installation.
- This repository is scaffolded and compiled successfully as of setup.
- Main app entry is in `src/app/page.tsx`.

## Deployment

Deploy to Vercel or GitHub Pages when ready. Suggested domain: `lightestdark.dev` or your custom domain.
